Отладка

1) Для вывода данных используется файл log.txt, который будет находиться в папке build, после сборки проекта. Чтобы включить логирование данных, нужно в Utils.pas раскомментировать строчку:
	```Pascal
	{$undef DEBUG}
	```
	Для вывода в файл, используйте встроенные функции: logln(str) - вывод строки с переносом на новую строку.


Особенности работы с языком PascalABC.Net

1) Создавая класс, необходимо создать и конструктор. Без него создать объект не получиться. Конструктора по умолчанию походу нету. Ошибка будет времени исполнения, а не компиляции.
